//サンプル

[MsgLayer visible=true]

// 30fps,800*600を想定してる。
// 多分間違えたコマンドを打っても何もしないだけでエラーにはならないと思う。
// 不満要望があればスカイプで言って


●プラグインを初期化する。[l][r]


	[InitMM]


●oggファイルをidに紐づけして読み込む。[l][r]
// url: oggの場所 必須
// id: 音楽にidを付ける。以降idを指定して再生などの操作をする。
//			初回は必須だが、省略した場合はこのコマンド以前に設定したidの値になってる。
//			例えば1曲しか操作しない場合はload時のみidを指定して、以降は省略できる。


	[MM mode=load url="test.ogg" id=primary]


●それを再生する。[l][r]
// volume: 再生時音量 0~1 省略すると1[r]
// time: フェードイン（二次関数的）にかける時間 単位ms 省略すると0[r]
// id: 操作するid 省略すると以前設定したid


	[MM mode=play volume=0.2 time=1000]


●音量を変える。[l][r]
// volume: 変更先の音量 0~1 省略すると1
// time: 変化(二次関数的)にかける時間 単位ms 省略すると0
// id: 操作するid 省略すると以前設定したid


	[MM mode=adjust volume=0.8 time=2000]


●一時停止する。[l][r]
// time: 変化(二次関数的)にかける時間 単位ms 省略すると0
// id: 操作するid 省略すると以前設定したid


	[MM mode=pause time=2000]


●再開する。[l][r]
// time: 変化(二次関数的)にかける時間 単位ms 省略すると0
// id: 操作するid 省略すると以前設定したid


	[MM mode=resume volume=0.8 time=2000]


●音楽を止める。[l][r]
// time: フェードアウト（二次関数的）にかける時間 単位ms 省略すると0
// id: 操作するid 省略すると以前設定したid


	[MM mode=stop time=1000]


●読み込んだ音楽を破棄する。もし再生中だったらブツ切りになる。[l][r]
// id: 操作するid 省略すると以前設定したid


	[MM mode=unload]


end
